# โค้ดเก่า (Version 1.0)

โฟลเดอร์นี้เก็บโค้ดเวอร์ชันเก่าของโปรเจกต์

## Tech Stack เดิม

### Frontend
- React 18
- Vite 5
- React Router v6
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT

## โครงสร้าง

```
old/
├── src/              # Frontend (React + Vite)
├── server/           # Backend (Express.js)
├── public/           # Static files
└── ...
```

## หมายเหตุ

- โค้ดนี้ถูกย้ายมาจาก root directory
- สามารถอ้างอิงโค้ดเก่าได้ตลอดเวลา
- ไม่แนะนำให้รันโค้ดนี้โดยตรง
- ใช้เป็น reference สำหรับการ migrate

## โปรเจกต์ใหม่

โปรเจกต์ใหม่อยู่ที่:
- `../frontend/` - Next.js 15
- `../backend/` - NestJS 11

ดูรายละเอียดได้ที่ [../README.md](../README.md)
